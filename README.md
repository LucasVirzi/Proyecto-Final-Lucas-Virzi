Hola yo soy Lucas Virzi. En este proyecto individual mi objetivo es que el usuario pueda crear indefinidos objetos o individuos, dentro de tres clases distintas, mascotas, familiares y autos. 
Ademas logre crear otro tipo de funciones, pero no viene al caso. 
Paso 1:
Para inicializar mi programa lo primero que el usuario debe hacer es escribir en la terminal git init. 
Paso 2:
Lo siguiente seria clonar mi programa en el sistema local del usuario, escribiendo git clone seguido de la url de mi proyecto subido a github. 
Paso 3:
Verificar lo programado y escribir 	http://127.0.0.1:8000/ en el buscador del usuario para poder ver las funciones creadas.
De este punto en adelante el usuario puede elegir que funciones probar. El orden seria primero crear un objeto(Familiar) con la funcion mi-familia/alta. Luego el familiar aparecera en la lista y se puede editar la informacion, borrarlo y hasta crear mas familiares.
